Workflow

O workflow é um conjunto de ferramentas primariamente criadas pelo time de desenvolvimento Suckless que têm como principal objetivo a simplicidade de código. As ferramentas que estão atualmente sendo utilizadas em meu workflow são o DWM, o dmenu, o slstatus e o slock.

DWM(Dynamic Window Manager) é um Tiling Window Manager que tem como pontos fortes a pouca quantidade de código fonte(1000 linhas em sua versão original), a extensibilidade a partir da linguagem C e sua leveza.

O dmenu é um wrapper de entrada e saída que pode também ser utilizado como um application launcher em muitos Window Managers minimalistas.

O slstatus é um conjunto de funções que podem ser utilizadas para customizar a barra de status do DWM.

O slock é um bloqueador de tela minimalista que tenta otimizar a quantidade de recursos utilizados pelo sistema durante longos períodos de tempo.

ST

ST(Simple Terminal) é um emulador de terminal vindo do projeto Suckless que da mesma forma que o DWM foi pensado visando a simplicidade e extensibilidade.
